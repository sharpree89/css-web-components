Usage

The default <tag> has an indigo background-color, white font color, and a subtle border-radius.

Tags have the exact same functionality as normal buttons, the major difference that tags are smaller and thus better suited for specific scenarios.

default 
          
      <div>
        <tag>default</tag>
      </div>

Use tags to style and emphasize keywords related to an item, such as an article or a recipe, that enable searching or navigating your site or app.

Customizing Tags

Colors

Use the .color class to change a tag's background-color. indigo offers 11 different colors and 21 different color styles.

If you choose to manually adjust the color, most components not only have background-color, they also have box-shadow and border-color properties that will need to be adjusted.

More custom colors are being added soon, so stay tuned!

red orange gold green blue indigo purple pink grey black 
          
      <div>
        <tag class="red">red</tag>
        <tag class="orange">orange</tag>
        <tag class="gold">gold</tag>
        <tag class="green">green</tag>
        <tag class="blue">blue</tag>
        <tag>indigo</tag>
        <tag class="purple">purple</tag>
        <tag class="pink">pink</tag>
        <tag class="grey">grey</tag>

      </div>

Use the .color-light class to change a tag's background-color, font color, and border-color.

      </div>
Ghost Tags

Use the .ghost class for a transparent background-color, white font color, and white border-color.

Transparent tags are most effective when paired with high-contrast background colors or images that won't clash with the tag's text.

ghost ghost ghost 
          
      <div>
        <tag class="ghost">ghost</tag>
        <tag class="ghost med-tag">ghost</tag>
        <tag class="ghost lrg-tag">ghost</tag>
      </div>

Sizes

Tags come in three sizes: small (default), medium, and large.

Small: <tag>, Medium: .med-tag, Large: .lrg-tag

small medium large 
          
      <div>
        <tag class="purple">small</tag>
        <tag class="purple med-tag">medium</tag>
        <tag class="purple lrg-tag">large</tag>
      </div>

Styles

Square Tags

Use the .square class to give tags squared, sharp edges.

square square square 
          
      <div>
        <tag class="green-light square">square</tag>
        <tag class="blue square med-tag">square</tag>
        <tag class="indigo-light square lrg-tag">square</tag>
      </div>
Pill Tags

Use the .pill class for a pill-shaped tag with rounded edges.

round round round 
          
      <div>
        <tag class="red-light pill">round</tag>
        <tag class="orange pill med-tag">round</tag>
        <tag class="gold-light pill lrg-tag">round</tag>
      </div>
