Tags

The default <tag> is indigo with white text and slightly rounded edges.

default tag 
          
      <div>
        <tag>default tag</tag>
      </div>

Use tags to style and emphasize keywords that enable searching or navigating.

html5 css3 javascript ui / ux front-end web dev web design 
          
      <div>
        <tag class="black">html5</tag>
        <tag class="black">css3</tag>
        <tag class="black">javascript</tag>
        <tag class="black">ui / ux</tag>
        <tag class="black">front-end</tag>
        <tag class="black">web dev</tag>
        <tag class="black">web design</tag>
      </div>

Tags also work well as miniature buttons for tiny user interfaces.

Get Started Today! Sign Me Up No Thanks 
          
      <div>
        Get Started Today!
        <tag>Sign Me Up</tag>
        <tag class="grey">No Thanks</tag>
      </div>
Follow Me: Pinterest Twitter GitHub 
          
      <div>
        Follow Me
        <tag class="red">Pinterest</tag>
        <tag class="blue">Twitter</tag>
        <tag class="black">GitHub</tag>
      </div>

Customizing Tags

Anchor: Colors

Use the color class to change the background color of the tag.

red orange gold green blue indigo purple pink grey black 
          
      <div>
        <tag class="red">red</tag>
        <tag class="orange">orange</tag>
        <tag class="gold">gold</tag>
        <tag class="green">green</tag>
        <tag class="blue">blue</tag>
        <tag>indigo</tag>
        <tag class="purple">purple</tag>
        <tag class="pink">pink</tag>
        <tag class="grey">grey</tag>
        <tag class="black">black</tag>
      </div>

Use the color-light class for an inverted color scheme.

red orange gold green blue indigo purple pink grey black 
          
      <div>
        <tag class="red-light">red</tag>
        <tag class="orange-light">orange</tag>
        <tag class="gold-light">gold</tag>
        <tag class="green-light">green</tag>
        <tag class="blue-light">blue</tag>
        <tag class="indigo-light">indigo</tag>
        <tag class="purple-light">purple</tag>
        <tag class="pink-light">pink</tag>
        <tag class="grey-light">grey</tag>
        <tag class="black-light">black</tag>
      </div>
Anchor: Sizes

Tags have two additional size classes: big-tag and huge-tag.

default large jumbo 
          
      <div>
        <tag class="purple">default</tag>
        <tag class="purple big-tag">large</tag>
        <tag class="purple huge-tag">jumbo</tag>
      </div>
Anchor: Styles

Use the square class to give the tag squared, sharp edges.

squared squared squared 
          
      <div>
        <tag class="green-light square">squared</tag>
        <tag class="blue square big-tag">squared</tag>
        <tag class="indigo-light square huge-tag">squared</tag>
      </div>

Use the pill class to give the tag a pill shape with rounded edges.

rounded rounded rounded 
          
      <div>
        <tag class="red-light pill">rounded</tag>
        <tag class="orange pill big-tag">rounded</tag>
        <tag class="gold-light pill huge-tag">rounded</tag>
      </div>

Use the raised class to give the tag shadows and a raised effect.

raised raised raised 
          
      <div>
        <tag class="pink raised square">raised</tag>
        <tag class="pink raised square big-tag">raised</tag>
        <tag class="pink raised square huge-tag">raised</tag>
      </div>

Use the disabled class to indicate the tag is inactive...

yesterday today tomorrow 
          
      <div>
        <tag class="blue pill big-tag disabled">yesterday</tag>
        <tag class="blue pill big-tag">today</tag>
        <tag class="blue pill huge-tag disabled">tomorrow</tag>
      </div>

... or to put emphasis on a more preferred element.

Go Back Check Out 
          
      <div>
        <tag class="grey big-tag disabled">Go Back</tag>
        <tag class="green big-tag raised">Check Out</tag>
      </div>

Use the ghost class to give the tag a transparent look.

These are most effective when paired with high-contrast background colors or images that won't clash with the tag's text.

ghost tag ghost tag ghost tag 
          
      <div>
        <tag class="ghost">ghost tag</tag>
        <tag class="ghost big-tag">ghost tag</tag>
        <tag class="ghost huge-tag">ghost tag</tag>
      </div>
