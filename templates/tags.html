Tag Buttons

This page contains all the information you need about indigo's tag button styling options. Use the side navigation to jump to different sections throughout the page.

Usage

The default <tag> is indigo with white text and slightly rounded edges.

default tag 
          
      <div>
        <tag>default tag</tag>
      </div>

Use tags to style and emphasize keywords related to an item, such as an article or a recipe, that enable searching or navigating your site or app.

Customizing Tags

Colors

Use the .color class to change the background color of the tag.

red orange gold green blue indigo purple pink grey black 
          
      <div>
        <tag class="red">red</tag>
        <tag class="orange">orange</tag>
        <tag class="gold">gold</tag>
        <tag class="green">green</tag>
        <tag class="blue">blue</tag>
        <tag>indigo</tag>
        <tag class="purple">purple</tag>
        <tag class="pink">pink</tag>
        <tag class="grey">grey</tag>
        <tag class="black">black</tag>
      </div>

Use the .color-light class for an inverted color scheme that is consistent with the style of Secondary Buttons.

red orange gold green blue indigo purple pink grey black 
          
      <div>
        <tag class="red-light">red</tag>
        <tag class="orange-light">orange</tag>
        <tag class="gold-light">gold</tag>
        <tag class="green-light">green</tag>
        <tag class="blue-light">blue</tag>
        <tag class="indigo-light">indigo</tag>
        <tag class="purple-light">purple</tag>
        <tag class="pink-light">pink</tag>
        <tag class="grey-light">grey</tag>
        <tag class="black-light">black</tag>
      </div>

Sizes

Tags come in three sizes: default, medium, and large.

Default - <tag> - no additional class necessary.

Medium - .med-tag

Large - .lrg-tag

default medium large 
          
      <div>
        <tag class="purple">default</tag>
        <tag class="purple med-tag">medium</tag>
        <tag class="purple lrg-tag">large</tag>
      </div>

Styles

Square Tags

Use the .square class to give tags squared, sharp edges.

square square square 
          
      <div>
        <tag class="green-light square">squared</tag>
        <tag class="blue square med-tag">squared</tag>
        <tag class="indigo-light square lrg-tag">squared</tag>
      </div>
Pill Tags

Use the .pill class for a pill-shaped tag with rounded edges.

rounded rounded rounded 
          
      <div>
        <tag class="red-light pill">rounded</tag>
        <tag class="orange pill med-tag">rounded</tag>
        <tag class="gold-light pill lrg-tag">rounded</tag>
      </div>

Use the .raised class to give tags shadows and a raised, 3-dimensional effect.

raised raised raised 
          
      <div>
        <tag class="pink raised square">raised</tag>
        <tag class="pink raised square med-tag">raised</tag>
        <tag class="pink raised square lrg-tag">raised</tag>
      </div>
Ghost Buttons

Use the .ghost class to give tags a transparent background with a white border and white text.

These are most effective when paired with high-contrast background colors or images that won't clash with the tag's text.

ghost ghost ghost 
          
      <div>
        <tag class="ghost">ghost</tag>
        <tag class="ghost med-tag">ghost</tag>
        <tag class="ghost lrg-tag">ghost</tag>
      </div>
