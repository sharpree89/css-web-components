Standard Buttons

This page contains everything you need to know about indigo's standard buttons. Use the side navigation to skip to different sections.



Primary Buttons

The default <button> has an indigo background-color, white font color, and a subtle border-radius.

          
      <div>
        <button>primary</button>
      </div>

Use primary buttons to indicate a clear call-to-action to the user.

Secondary Buttons

Secondary buttons are recommended for call-to-action items of a lower priority.

When used alongside primary buttons, they are useful for drawing the user's attention to a more preferred or recommended action.

To use secondary buttons, add the .color-light class to a <button>. Click here to view all colors.

          
      <div>
        <button class="indigo-light">secondary</button>
      </div>

Input Buttons

Use input buttons in HTML forms or for activating scripts. The default <input type="submit">, <input type="reset">, and <input type="button"> all have an indigo background-color, white font color, and a subtle border-radius.

          
      <div>
        <input type="submit" value="input">
      </div>

Anchor Buttons

Anchor buttons are simply anchor tags that have <button> styling applied to them, and an arrow animation on hover.

To use an anchor button, add the class .btn-link to an anchor tag.

Note that for the animation to work, you must wrap the text within a <span>.

Anchor 
          
      <div>
        <a href="#0" class="btn-link"><span>anchor </span></a>
      </div>

Customizing Buttons

Colors

Use the .color class to change a button's background-color. indigo offers 11 different colors and 21 different color styles.

The .color class will apply to any button, whether it is <button>, <input>, or <a class="btn-link">.

If you choose to manually adjust the color, most components not only have background-color, they also have box-shadow and border-color properties that will need to be adjusted.

More custom colors are being added soon, so stay tuned!


          
      <div>
        <button class="red">button</button>
        <button class="orange">button</button>
        <button class="gold">button</button>
        <button class="green">button</button>
        <button class="blue">button</button>
        <input type="submit" value="input">
        <input type="submit" class="purple" value="input">
        <input type="submit" class="pink" value="input">
        <input type="submit" class="grey" value="input">
        <input type="submit" class="black" value="input">
      </div>

Colors can help convey different meanings depending on the context.


Use the .color-light class to change a button's background-color, font color, and border-color for an inverted color scheme.

The .color-light class will apply to any button, whether it is <button>, <input>, or <a class="btn-link">.

Ghost Buttons

Use the .ghost class for a transparent background-color, white font color, and white border-color.

The .ghost class will apply to any button, whether it is <button>, <input>, or <a class="btn-link">.

Ghost buttons are most effective when paired with high-contrast background colors or images that won't clash with the button's text.

          
      <div>
        <button class="ghost">ghost</button>
        <button class="ghost med-btn">ghost</button>
        <button class="ghost lrg-btn">ghost</button>
      </div>

Sizes

All button types come in three sizes: small (default), medium, and large.

Use the following classes for sizing up <button> and <input> - Medium: .med-btn, Large: .lrg-btn

Use the following classes for sizing up <a class="btn-link"> - Medium: .med-btn-link, Large: .lrg-btn-link.

Note that size classes are to be used in addition to the base class of the button.

large 
          
      <div>
        <button class="purple">small</button>
        <input type="submit" class="purple med-btn" value="medium">
        <a href="#0" class="purple btn-link lrg-btn-link"><span>large </span></a>
      </div>

Styles

Square Buttons

Use the .square class to give buttons squared, sharp edges.

The .square class will apply to any button, whether it is <button>, <input>, or <a class="btn-link">.

          
      <div>
        <button class="green-light square">squared</button>
        <button class="blue square med-btn">squared</button>
        <button class="indigo-light square lrg-btn">squared</button>
      </div>
Pill Buttons

Use the .pill class for a pill-shaped button with rounded edges.

The .pill class will apply to any button, whether it is <button>, <input>, or <a class="btn-link">.

          
      <div>
        <button class="red-light pill">round</button>
        <button class="orange pill med-btn">round</button>
        <button class="gold-light pill lrg-btn">round</button>
      </div>
