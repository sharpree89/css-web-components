Buttons

The default primary <button> is indigo with white text and slightly rounded edges.

          
      <div>
        <button>default button</button>
      </div>

Use primary buttons to indicate a call-to-action.

          
      <div>
        <button class="orange">buy now</button>
        <button class="black">submit</button>
        <button class="blue">register</button>
      </div>

Buttons also help users navigate through a site or app.

          
      <div>
        <button class="black">back</button>
        <button class="pink">home</button>
        <button class="black">next</button>
      </div>

Customizing Buttons

Anchor: Colors

Use the color class to change the background color of the button.

          
      <div>
        <button class="red">red</button>
        <button class="orange">orange</button>
        <button class="gold">gold</button>
        <button class="green">green</button>
        <button class="blue">blue</button>
        <button>indigo</button>
        <button class="purple">purple</button>
        <button class="pink">pink</button>
        <button class="grey">grey</button>
        <button class="black">black</button>
      </div>

Use the color-light class for an inverted color scheme (secondary buttons).

          
      <div>
        <button class="red-light">red</button>
        <button class="orange-light">orange</button>
        <button class="gold-light">gold</button>
        <button class="green-light">green</button>
        <button class="blue-light">blue</button>
        <button class="indigo-light">indigo</button>
        <button class="purple-light">purple</button>
        <button class="pink-light">pink</button>
        <button class="grey-light">grey</button>
        <button class="black-light">black</button>
      </div>

Use secondary buttons alongside primary buttons to help emphasize a preferred action.

          
      <div>
        <button class="orange-light">go back</button>
        <button class="orange">check out</button>
      </div>
          
        
          
      <div>
        <button>sign up today</button>
        <button class="indigo-light">no thanks</button>
      </div>
Anchor: Sizes

Buttons have two additional size classes: big-btn and huge-btn.

          
      <div>
        <button class="purple">default</button>
        <button class="purple big-btn">large</button>
        <button class="purple huge-btn">jumbo</button>
      </div>
Anchor: Styles

Use the square class to give the button squared, sharp edges.

          
      <div>
        <button class="green-light square">squared</button>
        <button class="blue square big-btn">squared</button>
        <button class="indigo-light square huge-btn">squared</button>
      </div>

Use the pill class to give the button a pill shape with rounded edges.

          
      <div>
        <button class="red-light pill">rounded</button>
        <button class="orange pill big-btn">rounded</button>
        <button class="gold-light pill huge-btn">rounded</button>
      </div>

Use the raised class to give the button shadows and a raised effect.

          
      <div>
        <button class="pink raised square">raised</button>
        <button class="pink raised square big-btn">raised</button>
        <button class="pink raised square huge-btn">raised</button>
      </div>

Use the disabled class to indicate the button is inactive...

          
      <div>
        <button class="blue pill disabled">yesterday</button>
        <button class="blue pill">today</button>
        <button class="blue pill disabled">tomorrow</button>
      </div>

... or to put emphasis on a more preferred element.

          
      <div>
        <button class="grey disabled">Go Back</button>
        <button class="green raised">Check Out</button>
      </div>

Use the ghost class to give the button a transparent look.

Ghost buttons are most effective when paired with high-contrast background colors or images that won't clash with the button's text.

          
      <div>
        <button class="ghost">ghost</button>
        <button class="ghost big-btn">ghost</button>
        <button class="ghost huge-btn">ghost</button>
      </div>
