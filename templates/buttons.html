Standard Buttons

This page contains all the information you need about indigo's standard button styling options. Use the side navigation to jump to different sections throughout the page.

Primary Buttons

The default <button> is indigo with white text and slightly rounded edges.

          
      <div>
        <button>primary button</button>
      </div>

Use primary buttons to indicate a clear call-to-action to the user.

          
      <div>
        <button class="orange">buy now</button>
        <button class="black">sign up today</button>
        <button class="green">download</button>
      </div>

Secondary Buttons

To use secondary buttons, add a .color-light class to a <button>.

Click here to view all colors.

Secondary buttons are recommended for call-to-action items of a lower priority.

Also, when used alongside primary buttons, they are useful for drawing the user's attention to a more preferred or recommended action.

          
      <div>
        <button class="green">get started</button>
        <button class="green-light">go back</button>
      </div>
          
        
          
      <div>
        <button class="indigo-light">not right now</button>
        <button>sign up today</button>
      </div>

Input Buttons

Use input buttons for form submission. The default <input type="submit"> is indigo with white text and slightly rounded edges.

          
      <div>
        <label for="email">Email Address</label>
        <input type="text" id="email" placeholder="fozzy@bear.com">
        <input type="submit" value="input button">
      </div>

Anchor Buttons

Anchor buttons are simply anchor tags that have button styling applied to them. Use them for navigation.

Anchor buttons come in two styles: primary - .btn-link and secondary - .btn-link-light.

Note that for the animation on hover to work, you must wrap the anchor text within a <span>.

Primary Anchor Secondary Anchor 
          
      <div>
        <a href="#0" class="btn-link"><span>primary anchor </span></a>
        <a href="#0" class="btn-link-light"><span>secondary anchor </span></a>
      </div>

Customizing Buttons

Colors

Use the .color class to change the background color of a primary button, input button, or anchor button.


          
      <div>
        <button class="red">button</button>
        <button class="orange">button</button>
        <button class="gold">button</button>
        <button class="green">button</button>
        <button class="blue">button</button>
        <button>button</button>
        <button class="purple">button</button>
        <button class="pink">button</button>
        <button class="grey">button</button>
        <button class="black">button</button>
      </div>

Colors can help convey different meanings depending on the context.

Sizes

Primary buttons and secondary buttons come in three sizes: default, medium, and large.

Default - <button> - no additional class necessary.

Medium - .med-btn

Large - .lrg-btn

These classes currently do not work with anchor buttons or input buttons.

          
      <div>
        <button class="purple">default</button>
        <button class="purple med-btn">medium</button>
        <button class="purple lrg-btn">large</button>
      </div>

Styles

Square Buttons

Use the .square class to give buttons squared, sharp edges.

          
      <div>
        <button class="green-light square">squared</button>
        <button class="blue square med-btn">squared</button>
        <button class="indigo-light square lrg-btn">squared</button>
      </div>
Pill Buttons

Use the .pill class for a pill-shaped button with rounded edges.

          
      <div>
        <button class="red-light pill">rounded</button>
        <button class="orange pill med-btn">rounded</button>
        <button class="gold-light pill lrg-btn">rounded</button>
      </div>
Raised Buttons

Use the .raised class to give buttons shadows and a raised, 3-dimensional effect.

          
      <div>
        <button class="pink raised square">raised</button>
        <button class="pink raised square med-btn">raised</button>
        <button class="pink raised square lrg-btn">raised</button>
      </div>
Ghost Buttons

Use the .ghost class to give buttons a transparent background with a white border and white text.

Ghost buttons are most effective when paired with high-contrast background colors or images that won't clash with the button's text.

          
      <div>
        <button class="ghost">ghost</button>
        <button class="ghost med-btn">ghost</button>
        <button class="ghost lrg-btn">ghost</button>
      </div>
