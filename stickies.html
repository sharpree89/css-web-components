Stickys


The default <sticky> is indigo with white text and slightly rounded edges.

default sticky 
        
          <div>
            <sticky>default sticky</sticky>
          </div>

Use stickys to alert users with a message, to display new information, or to emphasize a specific block of text.

Stickys are not recommended for call-to-action items or for navigating a site or app. Instead, use buttons or tags.

3 days left warning 1 new e-mail success clearance sale 
        
          <div>
            <sticky class="black">3 days left</sticky>
            <sticky class="red">warning</sticky>
            <sticky class="black">1 new e-mail</sticky>
            <sticky class="green">success</sticky>
            <sticky class="black">clearance sale</sticky>
          </div>


Customizing Stickys


Use the color class to change the background color of the sticky.

red orange gold green blue indigo purple pink grey black 
        
          <div>
            <sticky class="red">red</sticky>
            <sticky class="orange">orange</sticky>
            <sticky class="gold">gold</sticky>
            <sticky class="green">green</sticky>
            <sticky class="blue">blue</sticky>
            <sticky>indigo</sticky>
            <sticky class="purple">purple</sticky>
            <sticky class="pink">pink</sticky>
            <sticky class="grey">grey</sticky>
            <sticky class="black">black</sticky>
          </div>

Use the color-light class for an inverted color scheme.

red orange gold green blue indigo purple pink grey black 
        
          <div>
            <sticky class="red-light">red</sticky>
            <sticky class="orange-light">orange</sticky>
            <sticky class="gold-light">gold</sticky>
            <sticky class="green-light">green</sticky>
            <sticky class="blue-light">blue</sticky>
            <sticky class="indigo-light">indigo</sticky>
            <sticky class="purple-light">purple</sticky>
            <sticky class="pink-light">pink</sticky>
            <sticky class="grey-light">grey</sticky>
            <sticky class="black-light">black</sticky>
          </div>

Stickys have two additional size classes: big-sti and huge-sti.

default large jumbo 
        
          <div>
            <sticky class="purple">default</sticky>
            <sticky class="purple big-sti">large</sticky>
            <sticky class="purple huge-sti">jumbo</sticky>
          </div>

Use the square class to give the sticky squared, sharp edges.

squared squared squared 
        
          <div>
            <sticky class="green-light square">squared</sticky>
            <sticky class="blue-light square big-sti">squared</sticky>
            <sticky class="indigo-light square huge-sti">squared</sticky>
          </div>

Use the pill class to give the sticky a pill shape with rounded edges.

rounded rounded rounded 
        
          <div>
            <sticky class="red-light pill">rounded</sticky>
            <sticky class="orange-light pill big-sti">rounded</sticky>
            <sticky class="gold-light pill huge-sti">rounded</sticky>
          </div>

Use the raised class to give the sticky shadows and a raised effect.

raised raised raised 
        
          <div>
            <sticky class="pink raised square">raised</sticky>
            <sticky class="pink raised square big-sti">raised</sticky>
            <sticky class="pink raised square huge-sti">raised</sticky>
          </div>

Use the disabled class to indicate the sticky is inactive.

disabled enabled 
        
          <div>
            <sticky class="blue pill disabled">disabled</sticky>
            <sticky class="blue pill">enabled</sticky>
          </div>

Use the ghost class to give the sticky a transparent look.

These are most effective when paired with high-contrast background colors or images that won't clash with the sticky's text.

ghost sticky ghost sticky ghost sticky 
        
          <div>
            <sticky class="ghost">ghost sticky</sticky>
            <sticky class="ghost big-sti">ghost sticky</sticky>
            <sticky class="ghost huge-sti">ghost sticky</sticky>
          </div>
